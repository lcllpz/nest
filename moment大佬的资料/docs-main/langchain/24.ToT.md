`ToT`（Tree of Thoughts，思维树）把推理做成一棵树。每一步不只沿一条链往下写，而是有意生成多个候选中间结论，对它们评估或打分，再决定展开哪些分支、剪掉哪些明显差的路径。直觉上是在脑子里同时试几条路，再收敛到一条或少数几条继续走。

上一篇 `23.CoT` 是单链。本篇讲树形分叉何时值得上、预算怎么卡、评估器怎么接、以及本系列 `30.LATS` 专文怎么衔接。

## 和 `CoT` 差在哪里

`CoT` 强调一条可读的主线，从起点顺着走到答案。`ToT` 强调在关键节点上有宽度，同一深度可以有多个兄弟节点，每个兄弟代表一种不同的中间说法、子目标或策略。

没有评估与剪枝的 `ToT` 会很快变成指数爆炸。工程上要同时卡住三件事，包括每层分支上限、每一轮保留几条活路、整棵树最多扩展多少个节点。

常见手段包括下面这些。

- 每层最多 `k` 个候选，或只做一两层分叉再收回单链
- 束搜索只保留全局 top `b` 条路径
- 为深度与总调用预算设硬上限，用完即停

## 典型循环长什么样

可以用扩展、评估、选择三个动词理解一整轮，多轮叠起来就是树在长高。扩展是长出子节点，评估是给节点或边贴分数，选择是决定下一轮从哪些节点继续扩展（不一定只留一个，取决于你的束宽设置）。

![20260326235755](https://raw.githubusercontent.com/xun082/md/main/blogs.images20260326235755.png)

扩展阶段用模型对当前节点状态采样多条下一步，例如三种解题策略、三个子目标表述、三个不同的下一步工具组合。评估阶段可以是同一模型换 `prompt` 做批评与打分，也可以是规则、单元测试、外部检索命中率、人工抽检。选择阶段按分数排序，或按 `UCT` 一类公式在探索与利用之间折中，直到触发终止条件。

终止条件务必写进产品说明，否则搜索停不干净。常见几类如下。

- 任意叶子节点被评估为已足够回答用户，或任务完成标记为真
- 达到最大深度或最大节点数、或本轮预算内 `LLM` 调用次数用尽
- 连续若干轮全局最优分数不再上升，提前收敛（要配防抖，避免偶然波动误停）

## 搜索形状怎么选

按层扩（类似逐层 `BFS`）实现简单，每一轮对当前前沿上所有节点一起扩展，再全局排序取 top `b`，适合要并行试探、且希望各深度机会相对均衡的任务。

束搜索在每一层只保留分数最好的 `b` 条完整前缀，内存和日志更可控，适合长链任务但怕树炸开的情况。

偏深搜的写法会先沿最高分子节点往下钻，钻到头再回溯试次优，实现和调试都重一些，和 `UCT` 驱动的 `MCTS` 家族更接近，本系列在 `30.LATS` 里走的是这条线上的工程化版本。

无论哪种形状，都建议把当前前沿显式存在状态里，方便断点恢复和可视化，不要散落在隐式递归里。

## 评估器怎么设计才好用

纯语言模型打分，经常出现兄弟节点分数挤在一团、拉不开差距的情况，搜索就失去方向。能接上真实信号会稳很多，例如代码跑测试、数学题对标准答案、检索任务看是否命中指定文档、业务规则检查是否触红线。

分数建议归一化到固定区间（例如 0 到 1），并在 `prompt` 里写清维度，例如正确性、简洁性、是否离题、是否可执行。同一套口径要对同一层的兄弟节点公平，题干与上下文拼接方式要一致，否则搜索会系统性偏向某一种写法。

若成本和延迟敏感，可以分层评估。先用极短 `prompt` 或启发式把明显离谱的枝砍掉，再对幸存者做精细模型打分。也可以只对每层分数靠近 top 的少数节点调用重评估，避免对每个叶子写长篇评论。

注意评估器自己的偏差。它若讨厌某种合法风格（例如过于简短），整棵树都会朝啰嗦或朝某种模板塌缩，需要通过抽检和对抗样本来修。

## 和 `30.LATS` 专文怎么衔接

本系列在 `30.LATS`（Language Agent Tree Search）里写了一种把蒙特卡洛树搜索与反思打分接进语言 `Agent` 的路线，包含节点选择、`UCT`、反向传播等细节。概念上它和论文语境里的 `ToT` 同属在树上做搜索，但实现上更贴近需要工具反馈、可执行动作、以及多轮观察后再打分的场景。

若你当前任务主要是单次解题、没有复杂工具循环，可以先用简化版 `ToT`，浅层分叉加模型评估或规则，把总节点数和深度锁死。若任务要在多步工具调用中探索不同策略，并希望分数沿路径回传影响下一步探索，直接读 `30.LATS` 往往更省时间。

## 带预算与前沿的 TypeScript 骨架

示例里为省事把 `ChatSiliconFlow` 的默认密钥写在常量里，你落地时请改成环境变量注入，不要把密钥写进仓库。

下面是一段可单独落盘的 TypeScript 示例，与仓库根目录的 `tot.ts` 同步。依赖装齐、环境就绪后执行即可在终端看到与下面接近的输出顺序。它在上文 `totRound` 与 `totSearchUntil` 的循环上补了配置、扩展计数和简易判停，便于接到 `LangGraph` 节点或你自己的队列调度里。

终端输出大致按下面节奏出现。

- 整段搜索以 `[start] running ToT search` 起头
- 每一轮扩展先有 `expander:<节点 id>`，下面分两截打印推理流与正文流，然后是 `parsed ok (zod strict)` 和本次推理长度
- 批量评判时换 `judge:batch:...` 标签，节奏相同
- 命中 `terminal` 或节点预算用尽时会出现 `[done]` 和当前节点数
- 若保留思考汇总，会按步骤标签把整段推理再打印一遍
- 末尾给出 `winner` 的 id、从根到叶的路径行，以及可能的 `final_answer`

````typescript
import {
  BaseChatModel,
  type BaseChatModelCallOptions,
  type BaseChatModelParams,
  type LangSmithParams,
} from "@langchain/core/language_models/chat_models";
import { type BaseMessage, AIMessageChunk } from "@langchain/core/messages";
import { type CallbackManagerForLLMRun } from "@langchain/core/callbacks/manager";
import { ChatGenerationChunk, type ChatResult } from "@langchain/core/outputs";
import { ChatPromptTemplate } from "@langchain/core/prompts";
import { z } from "zod";

export interface ThoughtNode {
  id: string;
  parentId: string | null;
  content: string;
  depth: number;
  score?: number;
  finalAnswer?: string;
  terminal?: boolean;
}

export interface TotSearchConfig {
  maxDepth: number;
  maxChildrenPerNode: number;
  beamWidth: number;
  maxTotalNodes: number;
}

export interface TotSearchState {
  frontier: ThoughtNode[];
  allNodes: Map<string, ThoughtNode>;
  expandedCount: number;
  seq: number;
  bestNode: ThoughtNode | null;
}

export interface TotReasoningEntry {
  label: string;
  reasoning: string;
}

export type TotReasoningLog = TotReasoningEntry[];

export interface JudgeItem {
  content: string;
  score: number;
  terminal: boolean;
  final_answer: string;
}

export type Expander = (node: ThoughtNode) => Promise<string[]>;
export type BatchJudge = (items: ThoughtNode[]) => Promise<JudgeItem[]>;

function makeChild(
  parent: ThoughtNode,
  content: string,
  state: TotSearchState,
): ThoughtNode {
  state.seq += 1;
  const child: ThoughtNode = {
    id: `n-${state.seq}`,
    parentId: parent.id,
    content,
    depth: parent.depth + 1,
  };
  state.allNodes.set(child.id, child);
  state.expandedCount += 1;
  return child;
}

export function getPathToRoot(
  target: ThoughtNode,
  allNodes: Map<string, ThoughtNode>,
): ThoughtNode[] {
  const path: ThoughtNode[] = [];
  let cur: ThoughtNode | undefined = target;

  while (cur) {
    path.push(cur);
    cur = cur.parentId ? allNodes.get(cur.parentId) : undefined;
  }

  return path.reverse();
}

function pickBetterNode(
  a: ThoughtNode | null,
  b: ThoughtNode | null,
): ThoughtNode | null {
  if (!a) return b;
  if (!b) return a;
  return (b.score ?? -1) > (a.score ?? -1) ? b : a;
}

export async function totRound(
  state: TotSearchState,
  cfg: TotSearchConfig,
  expander: Expander,
  judgeBatch: BatchJudge,
): Promise<{ nextFrontier: ThoughtNode[]; winner: ThoughtNode | null }> {
  const roundCandidates: ThoughtNode[] = [];

  for (const parent of state.frontier) {
    if (parent.depth >= cfg.maxDepth) continue;
    if (state.expandedCount >= cfg.maxTotalNodes) break;

    const childContents = await expander(parent);
    const children: ThoughtNode[] = [];

    for (const text of childContents.slice(0, cfg.maxChildrenPerNode)) {
      if (state.expandedCount >= cfg.maxTotalNodes) break;
      children.push(makeChild(parent, text, state));
    }

    if (children.length === 0) continue;

    const judged = await judgeBatch(children);

    for (let i = 0; i < children.length; i += 1) {
      const node = children[i]!;
      const item = judged[i]!;

      node.score = clamp01(item.score);
      node.terminal = Boolean(item.terminal);
      node.finalAnswer = String(item.final_answer ?? "").trim();

      if (
        node.terminal &&
        node.finalAnswer &&
        !node.content.includes("最终答案:")
      ) {
        node.content = `${node.content}\n最终答案: ${node.finalAnswer}`;
      }

      state.bestNode = pickBetterNode(state.bestNode, node);
      if (node.terminal) {
        return { nextFrontier: [], winner: node };
      }

      roundCandidates.push(node);
    }
  }

  roundCandidates.sort((a, b) => (b.score ?? 0) - (a.score ?? 0));
  const nextFrontier = roundCandidates.slice(0, cfg.beamWidth);
  return { nextFrontier, winner: null };
}

export async function totSearchUntil(
  root: ThoughtNode,
  cfg: TotSearchConfig,
  expander: Expander,
  judgeBatch: BatchJudge,
): Promise<{ winner: ThoughtNode | null; state: TotSearchState }> {
  const state: TotSearchState = {
    frontier: [root],
    allNodes: new Map([[root.id, root]]),
    expandedCount: 1,
    seq: 1,
    bestNode: root,
  };

  while (state.frontier.length > 0 && state.expandedCount < cfg.maxTotalNodes) {
    const { nextFrontier, winner } = await totRound(
      state,
      cfg,
      expander,
      judgeBatch,
    );
    if (winner) return { winner, state };
    state.frontier = nextFrontier;
    if (state.frontier.length === 0) break;
  }

  return { winner: state.bestNode, state };
}

  async *_streamResponseChunks(
    messages: BaseMessage[],
    options: Partial<SiliconFlowCallOptions>,
    _runManager?: CallbackManagerForLLMRun,
  ): AsyncGenerator<ChatGenerationChunk> {
    const controller = new AbortController();
    let idleTimer: ReturnType<typeof setTimeout> | undefined;
    const bumpIdleDeadline = () => {
      if (idleTimer !== undefined) clearTimeout(idleTimer);
      idleTimer = setTimeout(() => controller.abort(), this.timeoutMs);
    };

      bumpIdleDeadline();
      const res = await fetch(`${this.baseURL}/chat/completions`, {
        method: "POST",
        headers: this.buildHeaders(),
        body: JSON.stringify(this.buildRequestBody(messages, options, true)),

      let verbosePrintedContentHdr = false;
      let verbosePrintedToolHdr = false;

        bumpIdleDeadline();
        const choice = event.choices?.[0];
        const delta = choice?.delta ?? {};

        const reasoningDelta =
          typeof delta.reasoning_content === "string"
            ? delta.reasoning_content
            : "";
        const contentDelta = normalizeDeltaContent(delta.content);
        const toolCallDelta = Array.isArray(delta.tool_calls)
          ? delta.tool_calls
          : [];

        if (reasoningDelta) {
          fullReasoning += reasoningDelta;
          if (this.verboseStream) {
            if (!verbosePrintedReasoningHdr) {
              process.stdout.write("\n—— 推理 reasoning_content ——\n");
              verbosePrintedReasoningHdr = true;
            }
            process.stdout.write(reasoningDelta);
          }
        }

        if (contentDelta) {
          if (this.verboseStream) {
            if (!verbosePrintedContentHdr) {
              process.stdout.write("\n—— 正文 content ——\n");
              verbosePrintedContentHdr = true;
            }
            process.stdout.write(contentDelta);
          }
        }

        if (toolCallDelta.length > 0) {
          toolDeltas.push(...toolCallDelta);
          if (this.verboseStream) {
            if (!verbosePrintedToolHdr) {
              process.stdout.write("\n—— tool_calls ——\n");
              verbosePrintedToolHdr = true;
            }
            process.stdout.write(`${JSON.stringify(toolCallDelta)}`);
          }
        }

        if (!reasoningDelta && !contentDelta && toolCallDelta.length === 0) {
          continue;
        }

        const chunkMessage = new AIMessageChunk({
          content: contentDelta,
          additional_kwargs: {
            reasoning_content_delta: reasoningDelta,
            reasoning_content: fullReasoning,
            tool_calls_delta: toolCallDelta,
            tool_calls_raw: toolDeltas,
          },
          response_metadata: {
            finish_reason: choice?.finish_reason ?? null,
            usage: event.usage ?? {},
          },
        });

        yield new ChatGenerationChunk({
          text: contentDelta,
          message: chunkMessage,
          generationInfo: {
            finish_reason: choice?.finish_reason ?? null,
            usage: event.usage ?? {},
          },
        });
      }

      if (this.verboseStream) {
        process.stdout.write("\n");
      }
    } finally {
      if (idleTimer !== undefined) clearTimeout(idleTimer);
    }
  }
}

async function* iterateSSE<T>(
  stream: ReadableStream<Uint8Array>,
): AsyncGenerator<T> {
  const reader = stream.getReader();
  const decoder = new TextDecoder("utf-8");
  let buffer = "";

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;

    buffer += decoder.decode(value, { stream: true });

    while (true) {
      const boundaryIndex = buffer.indexOf("\n\n");
      if (boundaryIndex === -1) break;

      const rawEvent = buffer.slice(0, boundaryIndex);
      buffer = buffer.slice(boundaryIndex + 2);

      const dataLines = rawEvent
        .split("\n")
        .map((line) => line.replace(/\r$/, ""))
        .filter((line) => line.startsWith("data:"))
        .map((line) => line.slice(5).trim())
        .filter(Boolean);

      if (dataLines.length === 0) continue;

      const data = dataLines.join("\n");
      if (data === "[DONE]") return;
      yield JSON.parse(data) as T;
    }
  }

  const tail = buffer.trim();
  if (!tail) return;

  const dataLines = tail
    .split("\n")
    .map((line) => line.replace(/\r$/, ""))
    .filter((line) => line.startsWith("data:"))
    .map((line) => line.slice(5).trim())
    .filter(Boolean);

  for (const data of dataLines) {
    if (data === "[DONE]") return;
    yield JSON.parse(data) as T;
  }
}

function normalizeDeltaContent(content: unknown): string {
  if (typeof content === "string") return content;
  if (!Array.isArray(content)) return "";

  return content
    .map((item) => {
      if (typeof item === "string") return item;
      if (
        item &&
        typeof item === "object" &&
        typeof (item as { text?: unknown }).text === "string"
      ) {
        return (item as { text: string }).text;
      }
      return "";
    })
    .join("");
}

function toProviderMessage(message: BaseMessage): ProviderMessage {
  const roleType = message._getType();
  const content = serializeMessageContent(message.content);

  if (roleType === "system") {
    return { role: "system", content };
  }
  if (roleType === "ai") {
    return { role: "assistant", content };
  }
  if (roleType === "tool") {
    const tool_call_id =
      typeof (message as unknown as { tool_call_id?: unknown }).tool_call_id ===
      "string"
        ? (message as unknown as { tool_call_id: string }).tool_call_id
        : undefined;
    return { role: "tool", content, tool_call_id };
  }
  return { role: "user", content };
}

function serializeMessageContent(content: BaseMessage["content"]): string {
  if (typeof content === "string") return content;
  if (!Array.isArray(content)) return String(content ?? "");

  return content
    .map((part) => {
      if (typeof part === "string") return part;
      if (part && typeof part === "object") {
        const maybeText = part as { text?: unknown; type?: unknown };
        if (typeof maybeText.text === "string") return maybeText.text;
        return JSON.stringify(part);
      }
      return "";
    })
    .join("\n");
}

function clamp01(value: number): number {
  return Math.max(0, Math.min(1, value));
}

function stripMarkdownCodeFences(text: string): string {
  let s = text.trim();
  if (!s.startsWith("```")) return s;
  s = s.replace(/^```[a-zA-Z0-9_-]*\s*/, "");
  s = s.replace(/\s*```\s*$/, "");
  return s.trim();
}

function fixMalformedJsonRootKey(s: string): string {
  let t = s.trim();
  for (let k = 0; k < 8; k++) {
    const next = t
      .replace(
        /\{\s*"\{\s*"(children|items)"(\s*:)/g,
        (_, key: string, colon: string) => `{"${key}"${colon}`,
      )
      .replace(
        /\{\s*\{\s*"(children|items)"(\s*:)/g,
        (_, key: string, colon: string) => `{"${key}"${colon}`,
      );
    if (next === t) break;
    t = next;
  }
  return t;
}

function extractFirstBalancedJsonObject(s: string): string | null {
  const start = s.indexOf("{");
  if (start === -1) return null;
  let depth = 0;
  let inString = false;
  let escaped = false;
  for (let i = start; i < s.length; i++) {
    const ch = s[i]!;
    if (escaped) {
      escaped = false;
      continue;
    }
    if (inString) {
      if (ch === "\\") escaped = true;
      else if (ch === '"') inString = false;
      continue;
    }
    if (ch === '"') {
      inString = true;
      continue;
    }
    if (ch === "{") depth++;
    else if (ch === "}") {
      depth--;
      if (depth === 0) return s.slice(start, i + 1);
    }
  }
  return null;
}

function parseModelJsonValue(raw: string): unknown {
  const tryOne = (x: string): unknown | null => {
    try {
      return JSON.parse(x) as unknown;
    } catch {
      return null;
    }
  };

  let s = fixMalformedJsonRootKey(stripMarkdownCodeFences(raw.trim()));
  const direct = tryOne(s);
  if (direct !== null) return direct;

  const balanced = extractFirstBalancedJsonObject(s);
  if (balanced) {
    const fixed = fixMalformedJsonRootKey(balanced);
    const r1 = tryOne(fixed);
    if (r1 !== null) return r1;
    const r2 = tryOne(balanced);
    if (r2 !== null) return r2;
  }

  const start = s.indexOf("{");
  const end = s.lastIndexOf("}");
  if (start !== -1 && end > start) {
    const r3 = tryOne(fixMalformedJsonRootKey(s.slice(start, end + 1)));
    if (r3 !== null) return r3;
  }

  throw new Error(`Invalid JSON after cleanup: ${raw.slice(0, 200)}`);
}

async function streamJsonObject<T>(
  model: ChatSiliconFlow,
  prompt: ChatPromptTemplate,
  vars: Record<string, unknown>,
  label: string,
  schema: z.ZodType<T>,
): Promise<{ value: T; reasoning: string }> {
  console.log(`\n[${label}] start`);
  const messages = await prompt.formatMessages(vars);
  const stream = await model.stream(messages, {
    response_format: { type: "json_object" },
  });

  let finalText = "";
  let finalReasoning = "";

  for await (const chunk of stream) {
    const text =
      typeof chunk.content === "string"
        ? chunk.content
        : serializeMessageContent(chunk.content);
    finalText += text;

    const reasoning =
      typeof chunk.additional_kwargs?.reasoning_content_delta === "string"
        ? chunk.additional_kwargs.reasoning_content_delta
        : "";
    if (reasoning) {
      finalReasoning += reasoning;
    }
  }

  const raw = parseModelJsonValue(finalText);
  const parsed = schema.parse(raw);
  console.log(`[${label}] parsed ok (zod strict)`);
  if (finalReasoning) {
    console.log(
      `[${label}] reasoning captured length=${finalReasoning.length}`,
    );
  }
  return { value: parsed, reasoning: finalReasoning };
}

const judgeItemSchema = z.object({
  content: z.string(),
  score: z.number(),
  terminal: z.boolean(),
  final_answer: z.string(),
});

function expandSchemaForMax(maxChildren: number) {
  return z
    .object({
      children: z.array(z.string().min(1)).min(1).max(maxChildren),
    })
    .strict();
}

function judgeSchemaForBatchSize(n: number) {
  return z
    .object({
      items: z.array(judgeItemSchema).length(n),
    })
    .strict();
}

const expandPrompt = ChatPromptTemplate.fromMessages([
  [
    "system",
    [
      "你是 ToT 扩展器。",
      "你要基于当前节点，生成多个不同但都合理的下一步候选思路。",
      "要求：短、清晰、可评估、彼此有区分度。",
      "不要直接输出解释，只返回 JSON。",
      "不要使用 markdown 代码块、不要用 ``` 包裹，从第一个 {{ 到最后一个 }} 仅此一段。",
      '格式: {{"children":["候选1","候选2","候选3"]}}',
    ].join("\n"),
  ],
  ["human", "当前节点:\n{node_content}\n\n最多生成 {max_children} 条候选。"],
]);

const judgePrompt = ChatPromptTemplate.fromMessages([
  [
    "system",
    [
      "你是 ToT 批量评估器。",
      "你需要对多个候选节点同时做判断。",
      "对每个候选给出：",
      "1. score：0 到 1，越高表示越接近正确最终答案。",
      "2. terminal：是否已经足以作为最终答案。",
      "3. final_answer：如果 terminal=true，给出简洁最终答案，否则给空字符串。",
      "重要：只有当 final_answer 能直接回答原始问题根节点时，才允许 terminal=true。",

      "必须按输入顺序返回。",
      "不要解释，只返回 JSON。",
      "不要使用 markdown 代码块、不要用 ``` 包裹，从第一个 {{ 到最后一个 }} 仅此一段。",
      '格式: {{"items":[{{"content":"候选文本","score":0.88,"terminal":false,"final_answer":""}}]}}',

  ["human", "原始问题根节点:\n{root_question}\n\n候选节点列表:\n{items_json}"],
]);

function createExpander(
  model: ChatSiliconFlow,
  cfg: TotSearchConfig,
  reasoningLog?: TotReasoningLog,
): Expander {
  return async (node) => {
    if (node.depth >= cfg.maxDepth) return [];
    const label = `expander:${node.id}`;
    const { value: parsed, reasoning } = await streamJsonObject(
      model,
      expandPrompt,
      {
        node_content: node.content,
        max_children: cfg.maxChildrenPerNode,
      },
      label,
      expandSchemaForMax(cfg.maxChildrenPerNode),
    );
    if (reasoningLog && reasoning.trim()) {
      reasoningLog.push({ label, reasoning });
    }

    return parsed.children
      .map((item) => item.trim())
      .filter(Boolean)
      .slice(0, cfg.maxChildrenPerNode);
  };
}

function createBatchJudge(
  model: ChatSiliconFlow,
  rootQuestion: string,
  reasoningLog?: TotReasoningLog,
): BatchJudge {
  return async (items) => {
    const label = `judge:batch:${items.map((item) => item.id).join(",")}`;
    const { value: parsed, reasoning } = await streamJsonObject(
      model,
      judgePrompt,
      {
        root_question: rootQuestion,
        items_json: JSON.stringify(
          items.map((item) => ({
            id: item.id,
            content: item.content,
            depth: item.depth,
            parentId: item.parentId,
          })),
          null,
          2,
        ),
      },
      label,
      judgeSchemaForBatchSize(items.length),
    );
    if (reasoningLog && reasoning.trim()) {
      reasoningLog.push({ label, reasoning });
    }

    return parsed.items;
  };
}

async function main(): Promise<void> {
  const root: ThoughtNode = {
    id: "n-1",
    parentId: null,
    content: "问题: 12 \* 13 + 9 = ? 给出简洁结论。",
    depth: 0,
  };

  const cfg: TotSearchConfig = {
    maxDepth: 4,
    maxChildrenPerNode: 3,
    beamWidth: 4,
    maxTotalNodes: 30,
  };

  const model = new ChatSiliconFlow();

  const reasoningLog: TotReasoningLog = [];
  const expander = createExpander(model, cfg, reasoningLog);
  const judgeBatch = createBatchJudge(model, root.content, reasoningLog);

  console.log("[start] running ToT search");
  const { winner, state } = await totSearchUntil(
    root,
    cfg,
    expander,
    judgeBatch,
  );

  console.log(`\n[done] nodes=${state.expandedCount}`);
  for (const entry of reasoningLog) {
    console.log(
      `\n========== 深度思考 ${entry.label} ==========\n${entry.reasoning.trim()}\n`,
    );
  }
  if (!winner) {
    console.log("no winner and no best node found");
    return;
  }

  const path = getPathToRoot(winner, state.allNodes);
  console.log(`winner=${winner.id}`);
  console.log("path:");
  for (const step of path) {
    console.log(
      `- [d=${step.depth}] score=${step.score ?? "-"} ${step.content}`,
    );
  }

  if (winner.finalAnswer) {
    console.log(`final_answer=${winner.finalAnswer}`);
  }
}

void main().catch((error) => {

});

这段示例把束宽、深度和整棵树节点上限写进状态，用一次模型调用扩出多条子思路，再用批量评估 `prompt` 给兄弟节点打分，用 `Zod` 钉死 JSON 形状，从而在演示里跑通扩枝、同批评分和某个节点标成 `terminal` 时的提前收束。要贴近生产，还要补失败重试与超时、配额与可观测性、访问统计，并把搜索状态、父链回溯与对话消息同步放进图编排或会话层，在单文件里堆这些反而拖累阅读。

## 适合与不适合

适合创意与策略多样、且你能定义什么叫更好的任务，例如多种算法思路、多种写作大纲、多种调试假设。也适合答案难以一步生成、但中间状态可以部分打分或部分验证的情况。产品上要可解释时，树路径比单条 `CoT` 更容易展示当时还试过哪些路。

不适合分支含义模糊、评估只能靠模糊语感打分的场景，那样搜索容易在噪声里空转，账单却照涨。也不适合延迟极敏感、且正确答案几乎唯一的简单问答，这时 `23.CoT` 通常够用。更不适合没有明确状态表示的任务，若节点里该装什么都说不清，先回到任务建模，不要先上树搜索。

## 小结

`ToT` 把推理从一条线变成一棵树，核心是控制分叉宽度、配好评估、并把终止与预算写死。再往上，若同一中间结论要被多条路径引用、或需要显式合并不同分支的结论，树结构不够表达，下一篇 `25.GoT` 专文讲图形态。
